# 📊 Business Analytics Internship Project

## Investment Preference Analysis using Python

This project was completed as part of the **Business Analytics Internship** at **Cognifyz Technologies**.

The objective of this project is to analyze an investment preference dataset to understand investor behavior, savings objectives, investment choices, information sources, and expected returns using Python for data analysis and visualization.

---

## 📌 Project Objectives

- Understand the dataset structure.
- Analyze participant demographics.
- Identify the most preferred investment avenues.
- Study reasons behind investment decisions.
- Analyze savings objectives.
- Explore common investment information sources.
- Estimate investment duration.
- Analyze investment return expectations.
- Perform correlation analysis on numerical variables.
- Generate business insights from the findings.

## 📊 Key Findings

- Mutual Funds emerged as one of the most preferred investment avenues.
- Retirement planning is the primary savings objective among participants.
- Financial Consultants are the most trusted source of investment information.
- Most participants prefer medium-term investments (1–5 years).
- The majority expect returns between **20%–30%**.
- Investment decisions are influenced by multiple financial factors rather than a single demographic characteristic.
